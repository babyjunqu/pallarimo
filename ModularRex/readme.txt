To compile and run modrex branch:
1. Get OpenSim source
2. Put this source under OpenSim/ModRex (or other folder which can point to trunk with ../..)
3. Run prebuild 2008
4. In visual studio, add existing projects to OpenSim solution: ModularRex.csproj, ModularRex.RexFramework.csproj and ModularRex.NHibernate.csproj
5. Compile, Run.
